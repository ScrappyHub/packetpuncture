# Artifact Formats

PacketPuncture produces several artifact types.

---

# Commitment Payload

JSON structure representing the event commitment.

Typical fields:

- event type
- timestamp
- device id
- session id
- metadata

---

# Signature Message

Canonical text representation of the payload used for signing.

---

# Detached Signature

Generated using OpenSSH signing.

Example extension:

.sig

---

# Receipt Log Entry

Append-only JSON entry containing:

- action
- timestamp
- sequence number
- previous log hash
- artifact hashes

---

# Transcript Bundle

Generated by deterministic runs.

Contains:

- execution summary
- artifact hashes
- verification outputs
